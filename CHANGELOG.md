# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive test suite with pytest
- GitHub Actions CI/CD workflows
- Pre-commit hooks for code quality
- Poetry-based dependency management
- Type hints with mypy
- Code formatting with ruff
- Contributing guidelines and Code of Conduct
- Comprehensive documentation structure

### Changed
- Migrated from setuptools to Poetry
- Updated project structure for better organization

## [0.1.0] - Initial Release

### Added
- Complete TestRail REST API coverage
- Multi-profile configuration support
- CSV import functionality for test cases
- Multiple output formats (JSON, table)
- Field filtering capabilities
- 18 command groups covering all TestRail resources
- Interactive configuration initialization
- Environment variable support
- Profile-based authentication

### Supported Commands
- Projects management
- Test suites and sections
- Test cases with CSV import
- Test runs and plans
- Test results and milestones
- Attachments handling
- User management
- Custom fields support
- Raw API access
